Refresher Course

Specificity

  • This is the quality or state of being specific.
  • In the case of CSS, the specificity of one or more competing CSS rules determines which rule should be applied to your markup.

Axioms

  • Avoid the !important property as much as possible. Or never use it.
  • Never use IDs for styling.
  • Do not use inline styling.
  • Never prefix ids or classes with elements.
